1. Download Xdebug https://gist.github.com/odan/1abe76d373a9cbb15bed
2. File nach xampp/php/ext kopieren
3. xampp/php/php.ini in Editor öffnen
4. output_buffering = Off
5. [Xdebug] Sektion einfügen
5. Apache und MySQL über Xampp Controlpanel neustarten
6. Visual Studio Code PHP Debug installieren
7. Firefox Xdebug Helper Addon installieren
	Danach erscheint in der der Adressleiste rechts ein Käfer
	diesen anklicken und auf Debug, im VS-Code auf den Debuger
